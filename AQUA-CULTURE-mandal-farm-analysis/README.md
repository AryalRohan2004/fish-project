# Mandal Farm — Aquaculture Intelligence System

## Problem
Fish mortality spikes unpredictably. Feed is wasted. Farmers react
too late. This project turns 1 year of farm sensor data into
actionable daily decisions.

## Dataset
365 days · 3 ponds · 21 parameters
Pangasius, Rohu, and Rohu-Bhakur Polyculture
Collected: Mandal Fish Farm, Nepal 2025

## Approach
1. **Data Pipeline** — cleaning, feature engineering, SQL storage
2. **EDA** — mortality patterns, oxygen risk, seasonal analysis
3. **Business Analysis** — pond efficiency, feed waste, cost ROI
4. **ML Prediction** — mortality risk score, feed optimization
5. **Dashboard** — Streamlit real-time monitoring
